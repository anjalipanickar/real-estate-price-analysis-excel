# Real Estate Price Analysis (Excel Project)

This project analyzes the price per square foot of 2BHK and 3BHK flats over time using Microsoft Excel.

## 📊 Features
- Pivot Table analysis
- Line chart comparing 2BHK vs 3BHK (2005–2025)
- Pie chart showing price distribution in 2025
- Key observations and insights written directly in Excel
- PDF export for clean presentation

## 📁 Files Included
- `Price_Analysis.xlsx` – main Excel file with charts and observations
- `Price_Analysis.pdf` – formatted export for submission
- `README.md` – this description

## 📌 Tools Used
- Microsoft Excel

## 📅 Time Period Analyzed
- 2005 to 2025

---

This is a beginner-friendly data analytics project created entirely in Excel.
